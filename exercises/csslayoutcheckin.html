ENGLISH 391C

Introduction to Web Design

Course Description

ENGLISH 391C: Introduction to Web Design is a course at UMass Amherst. The objective of the course is to teach students about Web Design using HTML5 and CSS to build their own personal web portfolios. The class is taught by Janine Solberg and Christina Sun.
